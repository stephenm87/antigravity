🚀 Quick Setup (5 minutes)

Create a Google Form so all students can submit at the same time from their own devices. Then import the results here to instantly synthesize class-generated guidelines.

Create a new Google Form

Go to forms.google.com → click Blank form.

Set the Form title & description
Copy-paste this as the Form title:
Socratic Seminar — Dos & Don'ts
Copy-paste this as the Form description:
We are building OUR class guidelines for Socratic Seminars — based on YOUR ideas. Think about what makes a discussion productive and respectful. What should we all commit to doing? What should we avoid? Your responses will be combined with the rest of the class to create a shared set of rules we all agree on.
Add 3 questions (all required)

Mark each question as Required so every student contributes.

Question 1 — Short answer, Required
Title: Your Name (first name or initials)
Question 2 — Paragraph, Required
Title: What should we DO in a Socratic Seminar?
Description: List at least 1 thing (the more the better). Put each idea on its own line. Think about: listening, speaking, evidence, respect, preparation.
Question 3 — Paragraph, Required
Title: What should we AVOID in a Socratic Seminar?
Description: List at least 1 thing to avoid. Put each idea on its own line. Think about: behaviors, habits, or attitudes that hurt discussion.
Share the Form with students

Click Send → copy the link → share via Google Classroom, or project the link/QR code on screen. Students fill it out on their own devices in 2–3 minutes.

Export responses to this tool

Once all students have submitted: in Google Forms, click Responses → Link to Sheets (the green Sheets icon). This opens a Google Sheet. Select all cells (⌘A on Mac / Ctrl+A on PC), copy (⌘C / Ctrl+C).

Synthesize!

Click the "Import & Synthesize" tab above → paste the data into the text box → click Synthesize. Your class guidelines appear instantly, ready to print or project.

👩‍🏫 How to Share with Other Teachers

This tool is a single HTML file that runs entirely in the browser — no internet, no accounts, no install needed. Here's how to share it:

Step 1 — Share two things:
  • This HTML file (socratic-seminar-survey.html) — send via email, AirDrop, Google Drive, or USB
  • Your Google Form — in Google Forms, click the 3-dot menu → "Make a copy", then share the copy link with colleagues
Step 2 — Each teacher does this:
  • Opens the Google Form copy, shares it with their own class
  • After students submit, opens Google Sheets, copies the response data
  • Double-clicks the HTML file to open it in Chrome/Safari/any browser
  • Goes to "Import & Synthesize" tab → pastes data → clicks Synthesize
  • Prints or projects their class's unique guidelines

💡 Each teacher gets guidelines built from their own students' ideas — nothing is shared between classes. The HTML file works offline and doesn't store any data on a server.
